# Local AI Assistant (Universal Offline)

## Features
- Works on Windows, Linux, macOS, and Termux.
- ML-powered NLP on desktops; rule-based fallback on Termux.
- Open apps, type text, search files.
- Fully offline operation.
- Auto-detects OS and applies correct setup.

## Commands
- `open <appname>` : Open an app.
- `type <text>` : Type text.
- `search file <filename>` : Search file.
- `exit` : Quit assistant.
